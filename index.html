Селекторы

Список селекторов:

  • По id "#" : Идентификатор определяет уникальное имя элемента, которое используется для изменения его стиля и обращения к нему через скрипты.

  • По классу ".class" : Классы применяют, когда необходимо определить стиль для одного или нескольких элементов веб-страницы.

  • По названию элемента "А" : Параметры стиля будут примененны ко всем обьектам с данным селектором

  • Комбинированные "A+B": Параметры стиля будут примененны ко всем обьектам с данной комбинацией селекторов

  • По вложенности "А В": При создании веб-страницы часто приходится вкладывать одни элементы внутрь других. Чтобы стили для этих элементов использовались корректно, помогут вложенные селекторы.
